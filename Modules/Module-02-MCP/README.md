# Module 02 – Model Context Protocol (MCP)

##  Overview

This module introduced me to the Model Context Protocol (MCP), a standardized way for AI assistants such as Claude to securely communicate with external tools and resources.

Instead of only answering questions, MCP allows AI to interact with approved tools, making it much more useful for cybersecurity workflows.

##  Real-World Application

In a Security Operations Center (SOC), analysts often use many different security tools.

With MCP, an AI assistant can securely work with approved tools to:

- Gather investigation data
- Analyze logs
- Search threat intelligence
- Generate reports
- Automate repetitive tasks

This allows analysts to spend more time investigating threats instead of performing repetitive manual work.

---

##  Key Takeaways

MCP is the bridge between AI and external tools.

It enables AI assistants to perform useful tasks while maintaining secure and controlled access.
